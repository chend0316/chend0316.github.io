# 流媒体协议

# 技术背景：一个直播的流程

  • 主播端：采样、编码、推流
  • 服务器端：接流、流处理/转码、分发
  • 观众端：拉流、解码、播放

编码、转码、解码是跟流媒体压缩技术相关的，这里不介绍了。推流、拉流是跟流媒体协议相关的，这里重点介绍。 分发网络是将视频预加载到边缘节点，观众端从边缘节点拉取，能降低服务器的压力。
