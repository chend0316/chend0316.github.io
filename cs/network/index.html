# 20世纪60年代以前：电话网络

这种技术被称为电路交换，需要给通信双方建立一条连接，这个连接是物理上真实存在的（例如一条电路）。每个实体占用固定的带宽，即便打电话时一个字不说也会占用整条连接的资源。这种网络是可预测的，数据沿着预定的路线传输。在通信结束后需要清理连接，以便把资源让给别人。
