# IOC 容器

控制反转 (Inverse of Control, IOC) 是软件工程领域的一种设计原则，用来降低代码耦合度，详见维基百科。

依赖注入 (Dependency Injection, DI) 是 IOC 的一种实现手段，可以通过 Java 反射 (Reflection) 实现依赖注入。

这个教程给出了使用 IOC 容器创建/管理 Bean 的关键步骤。
