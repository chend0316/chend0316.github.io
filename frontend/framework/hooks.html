# React Hooks & Vue Composition

Hooks 解决了：

  • 实现关注点分离，业务代码不用被生命周期割裂
  • 让函数式组件有内部状态、可以执行副作用
  • 以往高阶组件会导致嵌套地狱，Hooks 没这个问题
