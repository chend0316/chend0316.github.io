# 二叉树遍历

先序遍历由于是尾递归，所以很容易改成迭代。 中序遍历改迭代比较难，主要方法有颜色标记法、左臂下探法。 后序遍历改迭代更难，我没研究。

二叉树遍历改迭代感觉没啥意义，有精力还不如先练其它题目。

# 并查集

并查集也是一种新型数据结构，教材中很少提，各语言也没有内置。

经典并查集的实现不难，需要熟练到能在白板上手写。并查集的深度决定其优劣，所以并查集有2种优化办法：rank优化、路径压缩。

  • 200. 岛屿数量
  • 547. 朋友圈
