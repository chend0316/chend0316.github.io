# DP 专题

DP 初始状态有两种做法：数组空间多开一个、循环内单独判断。不同场合下代码简洁度不一样，例如1143. 最长公共子序列在循环内单独判断就很挫。

1143 循环内单独判断 
class Solution:
    def longestCommonSubsequence(self, a: str, b: str) -> int:
        if not a or not b: return 0
        N, M = len(a), len(b)
        dp = [[0]*M for _ in range(N)]
        for i in range(N):
            for j in range(M):
                if i == 0 and j == 0: dp[i][j] = 1 if a[i] == b[j] else 0
                elif i == 0: dp[i][j] = 1 if a[i] == b[j] else dp[i][j-1]
                elif j == 0: dp[i][j] = 1 if a[i] == b[j] else dp[i-1][j]
                elif a[i] == b[j]: dp[i][j] = dp[i-1][j-1] + 1
                else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
        return dp[N-1][M-1]

1143 数组空间多开一个 
class Solution:
    def longestCommonSubsequence(self, a: str, b: str) -> int:
        if not a or not b: return 0
        n, m = len(a), len(b)
        dp = [[0]*(m+1) for _ in range(n+1)]
        for i in range(0, n):
            for j in range(0, m):
                if a[i] == b[j]: dp[i + 1][j + 1] = dp[i][j] + 1
                else: dp[i + 1][j + 1] = max(dp[i + 1][j], dp[i][j + 1])
        return dp[n][m]
