# 滑动窗口

滑动窗口和双指针一样，都是对暴力算法的优化，少枚举了很多区间。它在迭代的过程中维护L、R两个指针，每次迭代只会移动L或者R。

什么样的问题可以用滑动窗口来解决？

  • 新问题可在原问题的基础上“增量”计算得到
  • 可以根据当前问题的状态判断下一问题是移动L指针还是R指针（决策单调性）

为什么滑动窗口算法是正确的？对理论感兴趣的同学可以查阅「决策单调性」相关的资料。
